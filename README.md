**Virtual Internship for NFT Marketplace
This repository contains the code for a virtual internship project focused on developing a dynamic NFT marketplace website. The internship covers the following key components:
---------------------------------
Owl Carousel: Utilized Owl Carousel, a responsive and customizable carousel plugin, to create interactive carousels for displaying NFTs and authors.
---------------------------------
Dynamic Routing: Implemented dynamic routing to enable seamless navigation between different pages and ensure appropriate content display.
---------------------------------
Backend API: Integrated a backend API to fetch and display NFTs and authors dynamically.
---------------------------------
AOS Animations: Incorporated AOS (Animate On Scroll) animations to add engaging effects to various elements on the website.
